# Day 01 – Quick Notes

## What is System Design?

Planning how software components work together.

## Key Focus for Beginners

- Components
- Flow
- Responsibility of each part

## Java Context

- Controller → Service → Repository → Database
- Every Java backend app follows this pattern

## Interview One-Liner

"System design is about structuring software components and their interactions to solve a problem efficiently."

## Remember

- Small systems first
- Clarity > Complexity
- Think in boxes and arrows
